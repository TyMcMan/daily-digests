InquiTree: Evaluating AI Agents in the Scientific Inquiry Loop with Paper-Derived Research Trees

Formalizes scientific inquiry as interactive Research Trees — DAGs capturing hypothesis formulation, study design, result interpretation, and belief updating. Tests agents on a 30-paper pool, releases the open-access IT-18 subset, and finds agents exhibit "cognitive tunneling" in long-horizon tasks: critical judgment degrades as the research tree deepens.

Why it matters: A directly adoptable eval harness for testing whether a research-assist agent discovers rather than memorizes — a core distinction as LLM agents move into scientific workflows.

Memory Beyond Recall: A Dual-Process Cognitive Memory System for Self-Evolving LLM Agents (DCPM)

Proposes DCPM, a 6-layer memory hierarchy — raw inputs → atomic facts → belief trajectories → identity → domain schemas → latent intentions. A synchronous System 1 daytime writer handles live belief revision; an async System 2 nighttime engine runs schema induction. Goes far beyond per-turn RAG retrieval, enabling agents that revise their self-model over time. Authors at Tencent.

Why it matters: The most complete cognitive-architecture treatment of agent memory to date — immediately relevant to any long-lived agent deployment or personalization system that needs to change its beliefs as it learns.

AGENTSERVESIM: A Hardware-Aware Simulator for Multi-Turn LLM Agent Serving

Existing LLM serving simulators model stateless requests. Multi-turn agentic workloads interleave model calls with tool invocations, requiring stateful execution, cross-turn KV-cache management, and routing logic during tool-induced gaps. AGENTSERVESIM models these faithfully to enable capacity planning, scheduler design, and SLA analysis for production agent deployments. From the University of Central Florida.

Why it matters: Fills the gap for infrastructure engineers planning production agent serving stacks — a direct input to hardware sizing, batching policy design, and cost modeling.

SIGIL: Subtle Injection for Ground-Truth Inference of LLM Training Data

Embeds imperceptible canary sequences — five strategies: lexical-rare, lexical-phrase, syntactic, semantic, code-pattern — into protected text or code. Any LLM trained on those documents then exhibits statistically detectable behavioral signatures. Introduces a Membership Inference Score (MIS) grounded in Neyman-Pearson hypothesis testing with formal false-positive-rate control.

Why it matters: A provable, practical mechanism for content owners to detect unauthorized use of their data in LLM training — directly applicable to IP enforcement, copyright litigation, and data governance audits.
