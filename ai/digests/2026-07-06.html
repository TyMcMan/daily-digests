Stop Comparing LLM Agents Without Disclosing the Harness

Benchmark evaluation of 8 popular LLM agent frameworks on 3 task suites reveals 15–30% accuracy variance across harnesses for identical underlying models. Tool set, retry policy, memory strategy, and prompt scaffolding each account for significant variance, rendering undisclosed-harness comparisons near-meaningless.

Why it matters: Calls for a "harness disclosure" norm in agent benchmarking — akin to model card requirements — that the community has been slow to adopt.
