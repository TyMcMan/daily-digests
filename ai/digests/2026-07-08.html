Couples a parallel semi-autoregressive draft backbone with confidence-scheduled verification that adapts block length per request based on prefix survival probability and engine throughput. Your Agent's Memories Are Not Its Own: Forged Reasoning Attacks on LLM Agent Memory

Introduces FARMA (Forged Amplifying Rationale Memory Attack), which poisons an agent's stored reasoning chains rather than its factual memory. Why it matters: As long-lived agents with persistent memory become standard infrastructure, memory poisoning is a frontline production security threat every agent builder needs to understand and defend against.

Latent Programming Horizons in Coding Agents

Logistic-regression probes on coding-agent hidden states can predict — ahead of the agent's own edits — whether evolving code parses, passes its test suite, reduces failing tests, or introduces regressions.
