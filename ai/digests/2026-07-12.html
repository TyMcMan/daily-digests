Remember When It Matters: Proactive Memory Agent for Long-Horizon Agents

Meta AI paper addressing "behavioral state decay" — where task-critical information (prior attempts, open subgoals, constraints) gets buried in long agent trajectories and stops influencing decisions. A plug-and-play proactive memory module monitors the trajectory and injects structured memory-grounded reminders only when needed, delivering +8.3 pp on Terminal-Bench 2.0 and +6.8 pp on τ²-Bench across both weak and strong base agents.
Why it matters: Drop-in module for any unmodified base agent — no weight changes, no prompt rewrites; directly improves reliability of long-horizon agentic task completion.

Unified video diffusion framework (SIGGRAPH 2026) that simultaneously handles reconstruction of asynchronous event-camera streams, future-frame prediction, and temporal interpolation in a single model.
