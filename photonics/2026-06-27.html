Q.ANT Runs Generative AI on Photonic Hardware at ISC High Performance 2026

Q.ANT demonstrated live generative AI inference running on its photonic processor at ISC High Performance 2026, performing LLM matrix-vector multiplication in the optical domain using the company's integrated photonic chip. The demonstration used a quantised language model workload and showed end-to-end output with the photonic compute unit handling the core multiply-accumulate operations. Why it matters: this is the first publicly demonstrated live LLM inference on a photonic processor at a major HPC conference — moving the field from simulation and lab demos to a conference-floor system demonstration; the key remaining questions are throughput, precision, and the overhead of the photonic-electronic interface, which Q.ANT did not fully disclose.

🔧 PIC Design & EDA

flexcompute/autophotonicdesign — Automated Photonic Component Design with AI-Assisted Inverse Design

Flexcompute released autophotonicdesign, an open-source Python library integrating Tidy3D FDTD simulation with AI-assisted inverse design for automated photonic component generation. Supports grating coupler, directional coupler, ring resonator, and MMI splitter design workflows, with a gdsfactory export pipeline for direct GDS-II output. Includes pre-trained surrogate models for common component classes to reduce full-wave simulation calls. Why it matters: combining Tidy3D's simulation accuracy with surrogate-model acceleration and gdsfactory export in a single open-source library closes a major gap in the PIC design automation stack — engineers using the Tidy3D/gdsfactory toolchain can now run AI-guided component optimisation without building the surrogate model pipeline themselves.

📄 Research & Labs

Ultralow-Loss Silicon Nitride Integrated Photonic Platform for Scalable Quantum Photonic Circuits (arXiv)

Researchers demonstrate a Si3N4 photonic platform achieving sub-0.1 dB/m propagation loss — a record for integrated photonic waveguides at telecom wavelengths — using optimised LPCVD deposition and annealing protocols compatible with 200 mm wafer processing. The platform supports multi-photon entanglement generation, integrated single-photon detection interfaces, and dense wavelength-division multiplexing for scalable quantum circuit integration. Why it matters: propagation loss at this level enables multi-metre effective optical paths on-chip, making large-scale quantum photonic circuits with high gate fidelity feasible on a foundry-accessible Si3N4 process — this is a materials milestone for integrated quantum photonics teams using standard CMOS-compatible platforms.

Light-Driven Magnetic Memory Switches 1,000× Faster Than Conventional MRAM — QST / Applied Physics Letterscatch-up

Japan's QST demonstrated magnetic memory switching via single ultrashort laser pulses — no electric current required — at approximately 1,000× faster switching speeds than conventional electrically-driven MRAM with lower heat generation per operation. The underlying paper was published in Applied Physics Letters on June 8, 2026. The laser-driven mechanism exploits ultrafast demagnetisation via the inverse Faraday effect, enabling sub-picosecond magnetic state toggling. Why it matters: laser-writable non-volatile memory is a potential enabler of O-E-O-free data center architectures, allowing photonic interconnects to write directly into memory without an electrical bridge; currently at materials demonstration stage but the 1,000× speed advantage and direct photoelectric interface potential make this a priority item to track for future optical compute memory integration.

EPFL Chip-Scale Ultrafast Laser Matches Tabletop Femtosecond System Performancecatch-up

EPFL researchers demonstrated a chip-scale ultrafast laser fabricated on a silicon photonic platform that matches the pulse duration and coherence characteristics of tabletop femtosecond laser systems within a 1 cm² footprint. The device uses a novel nonlinear microresonator design to achieve sub-100 fs pulses with milliwatt-level peak power, sufficient for two-photon microscopy and optical coherence tomography applications. Why it matters: a foundry-compatible chip-scale femtosecond laser removes the largest barrier to integrating ultrafast optics into handheld and implantable medical imaging devices — and creates a path to on-chip nonlinear optical processing (frequency comb generation, LIDAR) without free-space beam paths.
